##oaosidl-site

Jekyll website repository for the OAOSIDL.

##Requirements
###Jekyll
```
gem install jekyll
```

###Twitter Bootstrap
```
cd res/
wget http://twitter.github.com/bootstrap/assets/bootstrap.zip
unzip bootstrap.zip
rm bootstrap.zip
```
